# web-puri-project

## 概要
このリポジトリはプリ機をWebアプリとして動かすことを目的に作成されました。
![アプリアイコン](assets/icons/icon-original.png)

## 技術スタック
- HTML
- CSS
- JavaScript
- TypeScript
- React
- C++ (WebAssembly)
- Web Worker
- WebAssembly
- Web Share API
- PWA
- Service Worker
- Dexie.js
- Selfie Segmentation (MediaPipe)
- COCO-SSD (TensorFlow.js)
- Face Landmarks Detection (TensorFlow.js)
- Fabric.js
- JSZip
- FileSaver.js
- Marked.js
