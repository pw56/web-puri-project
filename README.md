# web-puri-project

## 概要
このリポジトリはプリ機をWebアプリとして動かすことを目的に作成されました。

## 使用技術・言語
- HTML  
- CSS  
- JavaScript  
- React(TypeScript)  
- TensorFlow.js  
- WebGL  
- C言語(WebAssembly)  
- Web Worker

## 開発環境
- マシン:
> **Chromebook**  
> **GitHub Codespaces 2-core**

- エディタ:
> **Visual Studio Code**  
> **Cursor**

- コンパイラ:
> **React:** React Compiler  
> **WebAssembly:** Emscripten

- コーディングAI:
> **GPT-4.1**  
> **Claude Sonnet 3.5**  
> **Claude 4 Sonnet**

- デザイン:
