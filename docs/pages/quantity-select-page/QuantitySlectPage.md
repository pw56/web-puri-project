# QuantitySlectPage

## Overview
プリの撮影枚数を選択する。
撮影モードとプリモード選択時のどちらでもこのページに遷移。

## Elements

### PageTitle
| 項目  | 説明  |
| ------------ | ------------ |
| Overview  | ページタイトルを表示。  |

### QuantitySelector
| 項目  | 説明  |
| ------------ | ------------ |
| Overview  | 撮影するプリの枚数を入力エリア。  |

#### PuriCount
| 項目  | 説明  |
| ------------ | ------------ |
| Overview  | 撮影するプリの枚数を入力。  |

#### StartShootingButton
| 項目  | 説明  |
| ------------ | ------------ |
| Overview  | 撮影を開始。  |
| Behavior  | `SelftShootPage`に遷移。  |
