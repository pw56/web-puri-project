# Title
| 項目  | 説明  |
| ------------ | ------------ |
| Overview  | タイトルを表示するコンポーネント。  |
| Behavior  | ページ読み込み完了時から開始。`Sample1`→`Sample2`→`Heart1`→`Heart2`→`AppName`の順でフェードイン。  |
| Notes  | フェードインは各ページの`opacity`を`0`から`1`へ、`duration`は仕様書通り。 |

## NormalTitle
| 項目  | 説明  |
| ------------ | ------------ |
| Overview  | 通常モード時のタイトルをグループ化するコンポーネント。  |
| Behavior  | アニメーションはない。通常モード時(`normal`)に中身をそのまま返す。  |

### Sample1
| 項目  | 説明  |
| ------------ | ------------ |
| Overview  | アプリの使用イメージの画像。  |
| Behavior  | 0.5秒かけてフェードイン。  |
| Notes  | `sample-1.svg`を表示する。  |

## CustomTitle
| 項目  | 説明  |
| ------------ | ------------ |
| Overview  | カスタムモード時のタイトルを表示するコンポーネント。  |
| Behavior  | カスタムモード時(`custom`)に画像を表示する。アニメーションはフェードインのみ。設定ファイル(`puri.config.json`)でカスタマイズできる。  |
