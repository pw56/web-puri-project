# テキスト
- 種類: テキスト
- 機能: 指定されたテキストを表示する。
- 表示:
> 指定がない場合、ページ全体で設定されているフォント、テキスト要素の初期値のサイズ、テキスト要素の初期値の色で表示。
> ボタンの両端が半円になる程度の丸みを帯びている。  
> ![テキスト全種類](Text-all.png)
- 属性:
  - type
> テキストの種類(`text-plain`, `text-value`など)を指定。
> 
> ```xml
> <Text type="text-plain"/>
> ```
> 
  - font-size
> テキストのサイズを指定。
> 
> ```xml
> <Text font-size="20rem"/>
> ```
>
  - height
> テキストの高さ(縦方向の大きさ)を指定。
> 
> ```xml
> <Text height="20rem"/>
> ```
>
  - width
> テキストの横幅(横方向の大きさ)を指定。
> 
> ```xml
> <Text width="20rem"/>
> ```

## テキスト プレーン
- 属性: `plain`
- 表示:
> 指定された色の文字のテキスト。指定がない場合はピンクの文字となる。  
> ![text-plain](text-plain.png)

## テキスト 数値
- 属性: `value`
- 表示:
> 指定された色の文字、背景、縁のテキスト。
> 指定がない場合はピンクの文字と縁となる。
> CSSの`box-sizing`プロパティと`border-box`を使用して、ボタンの縁を内側に収まるようにする。  
> ![text-value](text-value.png)
