# PuriSelect
- 機能: 選択肢を表示し、選択項目が選択されたら登録された関数を実行する。

## Props:
| Prop  | 型  | 初期値  | 説明  |
| ------------ | ------------ | ------------ | ------------ |
| height  | number  | 自動  | 選択肢の高さ(縦方向の大きさ)を指定。  |
| width  | number  | 自動  | 選択肢の横幅(横方向の大きさ)を指定。  |
|   |   |   |   |
|   |   |   |   |
|   |   |   |   |

- 属性:
  - text
> 選択肢に表示するテキストを指定。
> 
> ```xml
> <Select text="りんご"/>
> ```

  - image
> 選択肢に表示する画像を指定。
> 
> ```xml
> <Select image="stars.png"/>
> ```

  - max-column
> 表示する選択肢の横方向の最大の個数を指定。
> 収まりきらない選択肢は、次の行に右から順に表示。
> 
> ```xml
> <Select max-column="5"/>
> ```

  - column
> カラーパレットの場合に有効。
> 表示する選択肢の横方向の個数を指定。
> 収まりきらない選択肢が発生した場合は非表示。
> 
> ```xml
> <Select column="5"/>
> ```

  - row
> カラーパレットの場合に有効。
> 表示する選択肢の縦方向の個数を指定。
> 収まりきらない選択肢が発生した場合は非表示。
> 
> ```xml
> <Select row="5"/>
> ```

## 選択肢 アクセントカラー
- 属性: `pink`
- 表示:
> アクセントカラーの文字のテキスト。メインカラーの背景。
> マウスホバー時にアクセントカラーの部分は薄い色に変化するアニメーションを表示。
> 選択時に、選択された選択肢の見た目を変更。
> 選択時の変化は、テキストはメインカラー、背景はアクセントカラーにする。
> ![select-pink](select-pink.png)
> ![select-pink-image](select-pink-image.png)

## 選択肢 メインカラー
- 属性: `white`
- 表示:
> メインカラーの文字、アクセントカラーの背景、アクセントカラーの縁のテキスト。
> CSSの`box-sizing`プロパティと`border-box`を使用して、ボタンの縁を内側に収まるようにする。  
> マウスホバー時にメインカラーの部分はアクセントカラーがかったメインカラーに変化するアニメーションを表示。
> 選択時に、選択された選択肢の見た目を変更。
> 選択時の変化は、テキストはアクセントカラー、背景はメインカラーにし、アクセントカラーの縁を表示。
> ![select-white](select-white.png)
> ![select-white-image](select-white-image.png)

## 選択肢 メニュー
- 属性: `menu`
- 表示:
> アクセントカラーの文字のテキスト。メインカラーの背景。
> CSSの`box-sizing`プロパティと`border-box`を使用して、ボタンの縁を内側に収まるようにする。  
> マウスホバー時にアクセントカラーの部分は薄い色に変化するアニメーションを表示。
> 選択時に、選択された選択肢の見た目を変更。
> 選択時の変化は、テキストはメインカラー、背景はアクセントカラーにする。
> ![select-menu](select-menu.png)


## 選択肢 カラーパレット
- 属性: `color`
- 表示:
> 選択時に、選択された選択肢の見た目を変更。
> 選択時の変化は、選択肢の上の中央にチェックマーク(color-selected.png)を表示。
> ![select-color](select-color.png)
