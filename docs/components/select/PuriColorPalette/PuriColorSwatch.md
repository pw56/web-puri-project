# PuriColorSwatch

## Props
| Prop  | 型  | 初期値  | 説明  |
| ------------ | ------------ | ------------ | ------------ |
| `color`  | `React.CSSProperties['color']`  | なし  | カラーピッカーの色を指定。  |

## Behavior
- ホバー時に、マウスカーソルをリンク(`pointer`)に変化
- 選択時に、サイズを一回り縮小

## Notes
- `PuriColorPalette`で内部的に使用され、エクスポートされない
