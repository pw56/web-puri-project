# fatch-text-file.ts

## fetchTextFile (関数)
- 概要  
> 指定されたパスからテキストファイルを取得し、その内容を文字列として非同期で返す。  
> HTTPレスポンスが正常でない場合は例外を投げる。

- 引数  
> 第1引数: 取得対象のファイルパス  
> ```typescript
> string // 第1引数
> ```

- 返り値  
> ファイルの内容（テキストデータ）  
> ```typescript
> Promise<string>
> ```

- 例外  
> HTTPレスポンスが正常でない場合、エラーメッセージを含む例外を返す。
