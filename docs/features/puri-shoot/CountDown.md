# CountDown

## Props
| Prop  | 型  | 初期値  | 説明  |
| ------------ | ------------ | ------------ | ------------ |
| `children`  | `number`  | 0  | ボタンに表示する数値を指定。  |
| `height`  | `React.CSSProperties['height']`  | 自動  | ボタンの高さを指定。  |
| `width`  | `React.CSSProperties['width']`  | 自動  | ボタンの横幅を指定。  |

## Hooks
| Hook  | 引数  | 返り値  | 説明  |
| ------------ | ------------ | ------------ | ------------ |
| `useSetNumber`  | `number`  | `void`  | 表示する数値を設定。  |

## Behavior
- カスタムフックで表示する数値を設定
- テキストのサイズは、コンポーネントのサイズに合わせて自動で調整
- コンポーネントのサイズは円になるように自動で調整

## Notes
- コンポーネントのサイズ設定は、数値が小さい方が適用される
