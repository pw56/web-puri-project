# 実装機能

以下が本アプリの実装機能となる

- 通常のカメラ撮影
- プリ機のような案内付き撮影モード(プリモード)
- デバイスの画像をアップロード
- 機械学習と数学的手法による背景除去と顔のパーツの抽出
- 数学的な手法による顔のパーツの加工
- ペン描き
- スタンプ
- パーツ加工(目・鼻・口・眉毛・輪郭)
- 補正(美肌加工・美白加工・ツヤ髪加工)
- 背景(壁紙)の変更
- 共有ボタン・印刷ボタンの設置
- 出力されたプリのアルバム機能(複数のPNG形式の画像 + 日付 + 位置情報 + IndexedDB)　※アップロードされたファイルの場合は使用不可能
