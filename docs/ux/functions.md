# 実装機能

以下が本アプリの実装機能となる

- 通常のカメラ撮影
- プリ機のような案内付き撮影モード(プリモード)
- デバイスの画像をアップロード
- 機械学習と数学的手法による背景除去と顔のパーツの抽出
- 数学的な手法による顔のパーツの加工
- ペン描き
- スタンプ
- パーツ加工(目・鼻・口・眉毛・輪郭)
- 補正(美肌加工・美白加工・ツヤ肌加工・ツヤ髪加工)
- 背景(壁紙)の変更
- フィルター加工(レトロ風・フィルム風・ぼかし加工など多数)
- フレームレイアウト切り替え ※アップロードされたファイルの場合は使用不可能
- 撮影・加工して出力されたプリのダウンロード・アプリ側の一時保存(JPEG形式)
- 共有ボタン・印刷ボタンの設置
